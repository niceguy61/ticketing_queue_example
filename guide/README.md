# 학습 가이드

이 폴더에는 티케팅 큐 시스템을 통해 메시지 큐 아키텍처를 학습하기 위한 강의 자료가 포함됩니다.

## 📚 목차

(준비 중)

## 🎯 학습 목표

1. **메시지 큐 기본 개념** 이해
2. **Redis ZSET**을 활용한 실시간 대기열 구현
3. **RabbitMQ**를 활용한 비동기 메시지 처리
4. **AWS SQS**를 활용한 클라우드 네이티브 아키텍처
5. **하이브리드 아키텍처** 설계 및 구현
