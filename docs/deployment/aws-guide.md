# AWS Deployment Guide

Guide for deploying the Ticketing Queue System to AWS.

## Overview

This guide covers deploying services to AWS using:
- ECR (Elastic Container Registry) for container images
- EC2 (Elastic Compute Cloud) for running services
- RDS (Relational Database Service) for PostgreSQL
- ElastiCache for Redis

## Prerequisites

- AWS Account
- AWS CLI configured
- Docker installed locally
